# project for Software for Mobile Devices

This is an android-based (API 28+) tracking, reminders application that has the following:
- overview of your activity (such as tasks completed)
- Option to add tasks, reminders
- link to your social media profile
- Material Design by Google
- Uses dynamic view in Android view (Recycler view)

How to use:
1. Clone (can use Android Studio)
2. Run in Android studio / ecclipse or equivalent
